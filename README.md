# SkyRent Drones

Drone rental application with integrated identity verification SDK.

## Project Structure

Monorepo with pnpm workspaces:

- `packages/identity-sdk` — Reusable identity verification SDK
- `apps/demo` — SkyRent Drones demo application

## Getting Started

### Prerequisites

- Node.js >= 18
- pnpm >= 9

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

### Lint

```bash
pnpm lint
pnpm lint:fix
```
